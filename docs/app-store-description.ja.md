# App Store 説明文 — 日本語 (4000 文字以下)

Zeed は、プライバシーを最優先する AI 統合モバイルブラウザです。
OpenRouter API key を自分で持ち込んで、300 以上の LLM (GPT, Claude,
Gemini, GLM, DeepSeek, Llama など) から好きなモデルを選べます。
記憶・ブックマーク・閲覧履歴は端末から出ない設計です。

なぜ Zeed か

- 自分の LLM を使う。OpenRouter 経由で、モデル提供者に直接料金を払い
  ます。ベンダーロックインなし、マネージド LLM のマージンなし。
  タスクごとに自由にモデルを切替えられます。

- 構造的なプライバシー。記憶・ブックマーク・タスク・チャット履歴・
  Context Map は端末にのみ保存されます。テレメトリはオプトインで、
  匿名の診断・使用イベントだけ (install、session start、daily heartbeat、
  feature カウンター、Agent 実行結果、クラッシュハッシュ)。閲覧 URL・
  ページ内容・チャット内容は一切送信されません。

- 1 つの入力欄に 3 つのモード。Auto は意図を推測。Ask はチャット。
  Search は検索クエリ。タップ 1 つで切替えられます。

- タブが記憶になる。Profile → TabGroup → Tab の階層を持ち、Agent は
  開いているタブと、同じグループで閉じたタブの両方を文脈にして考え
  ます。

- プライベートモードはゼロトレース。プライベート中は、記憶への書込
  み・Context Map 構築・Agent 実行・レコメンド生成・テレメトリ送信
  をすべて停止します。

- Markdown + zeed-graph 描画。チャット回答は完全な Markdown と、構造
  (関係・選択肢・決定木) を線で表示する独自フォーマット zeed-graph
  をインラインで表示します。

Agent ができること

Agent は現在のページを読み、以下が可能です。

- リンクのクリック、フォーム入力、画面遷移
- 新しいタブを開く、切替え
- リサーチ目的のときは Web 検索
- 現在のタブの読解と要約

明示確認なしにやらないこと

- フォーム送信、購入、メッセージ送信などは必ず確認ダイアログを挟み
  ます
- プライベートモード中は Agent を実行しません
- プライベートセッションのデータは一切保存しません

ロードマップ

- Cloud Autopilot (Path B): 長時間タスクをクラウドで代行する
  オプトイン機能。データ境界は厳格に守ります
- マルチデバイス同期: オプトイン + エンドツーエンド暗号化 (Phase 6)
- スケジュール実行 ("cron"): 朝の情報確認や週次チェックを自動で
  走らせる (Phase 6 以降)

要件

- OpenRouter API key (openrouter.ai で無料作成可能)
- iOS 16 以降
- チャット・Agent 動作にはインターネット接続が必要

プライバシー

- 収集しない: 閲覧 URL・ページ内容・チャット内容・記憶・ブックマーク・
  IP アドレス・メールアドレス
- 収集する (匿名・オプトイン): install ID (ランダム UUID)、アプリ
  バージョン、OS、daily heartbeat、session start、feature カウンター
  (固定 allowlist)、Agent 実行結果 (成功 / ステップ数 / 終了理由)、
  クラッシュハッシュ
- テレメトリは初期状態で OFF
- 詳細: https://zeed.run/privacy
